PROJECT DESCRIPTION
------------------
A comprehensive mobile application designed to streamline the campus
placement process. Students can check their eligibility for various
companies based on their academic performance, while administrators
can manage company criteria and view placement statistics.

KEY FEATURES
-----------

STUDENT MODULE:
✓ Secure login system
✓ Profile management (Name, Register Number, Department, CGPA, Backlogs)
✓ Company listing with detailed criteria
✓ Real-time eligibility checking
✓ Detailed eligibility status with reasons
✓ Interactive dashboard with statistics
